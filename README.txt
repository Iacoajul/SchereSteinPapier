IDs
gameContainer: 
The container element where the game UI is created.
resultDisplay: 
The element where the results of each round are displayed.
roundCount: 
The element where the current round number is displayed.
currentRound: 
The span element inside roundCount that displays the current round number.
scoreBoard: 
The element where the scores of the participants are displayed.
winnerDisplay:
The element where the winner is displayed on win.html (currently)

Wahrscheinlich nicht stylebare IDs
score_{participant.id}:
 The span element inside scoreBoard that displays the score of each participant.
 Use scoreBoard instead
selectionArea_{participant.id}: 
The section element for each participant's selection area.
Use selctioArea class instead

Classes
selectionArea: 
The class applied to each participant's selection area.
